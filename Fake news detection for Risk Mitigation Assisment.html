Prepare text data for the models by cleaning and normalizing data.¶

For cleaning, tokenize text and split it into individual words, convert text to lowercase, remove punctuations, replace numbers by spaces, remove stopwords and lemmatize the text.

For normalization, convert multiple spaces to single spaces.

In [ ]:
tf_lemmatizer = WordNetLemmatizer()
tf_stopwords = set(stopwords.words('english'))

def Clean(text):
    text = str(text).lower()
    text = re.sub(f"[{string.punctuation}]", "", text)
    text = re.sub(r'\d+', ' ', text)
    text = re.sub(r'\s+', ' ', text).strip()
    tokens = word_tokenize(text)
    tokens = [tf_lemmatizer.lemmatize(w) for w in tokens if w not in tf_stopwords]
    return " ".join(tokens)

df_news['clean_text'] = df_news['text'].apply(Clean)
print(df_news[['clean_text', 'label']].head())

def compute(eval_pred):
    preds, labels = eval_pred
    preds = np.argmax(preds, axis=1)
    return {
        "accuracy": accuracy_score(labels, preds),
        "f1": f1_score(labels, preds, zero_division=0),
    }

Result Evaluation¶

Analyzing results from all 4 models:

TF-IDF has given highest accuracy of 0.99 for this dataset, amongst all other models. It was easy to execute and fastest.

Whereas, Word2Vec had lower accuracies as compared to TF-IDF:

W2V + ANN = 0.97 accuracy; W2V + RNN = 0.94

ANN model was quick as compared to RNN model as it took more number of epochs.

BERT model on the other hand exhausts lot of computational power and time due to GPU usage.

The results for this dataset on different models are quite contrasting to what usually happens. TF-IDF is the simplest amongst the all and usually shows lower performance but for this smaller dataset it proved to be more efficient and accurate.
